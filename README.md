## WAREHOUSE


How to compile:

```sh
javac -cp po-uilib.jar:. `find ggc -name "*.java"`

```

How to run:

```sh
java -cp po-uilib.jar:. ggc.app.App

```

How to run with import:
```sh
java -cp po-uilib.jar:. -Dimport=./example.import ggc.app.App

```
